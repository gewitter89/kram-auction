📘 API Документація

KRAM.UA REST API v1. Аутентифікація через Bearer Token (JWT).

Auth Lots Bids Users Messages Favorites Purchases Payment Notifications Reviews Admin System

🔐 Auth

POST/api/auth/registerno auth
Реєстрація нового користувача. Надсилає email для верифікації.
Body: { "username": "string", "email": "string", "password": "string", "firstName?": "string", "lastName?": "string", "phone?": "string", "city?": "string" }
Response 201: { "message": "Перевірте email для підтвердження", "token": "jwt", "user": { "id": 1, "username": "...", "email": "...", "firstName": "...", "lastName": "..." } }
POST/api/auth/loginno auth
Вхід за логіном або email. Встановлює csrf_token cookie.
Body: { "login": "string", "password": "string" }
Response 200: { "message": "Вхід успішний!", "token": "jwt", "user": { "id": 1, "username": "...", "email": "...", "firstName": "...", "lastName": "...", "rating": 4.5, "reviewsCount": 12, "city": "..." } }
GET/api/auth/meauth
Отримати профіль поточного користувача.
Headers: Authorization: Bearer <token>
Response 200: { "id": 1, "username": "...", "email": "...", "first_name": "...", "last_name": "...", "phone": "...", "city": "...", "bio": "...", "avatar": "...", "rating": 4.5, "reviews_count": 12, "created_at": "...", "is_verified": 1 }
PUT/api/auth/profileauth
Оновити профіль.
Body: { "firstName?": "string", "lastName?": "string", "phone?": "string", "city?": "string", "bio?": "string" }
Response 200: { "message": "Профіль оновлено" }
PUT/api/auth/passwordauth
Змінити пароль.
Body: { "currentPassword": "string", "newPassword": "string" }
Response 200: { "message": "Пароль змінено" }
POST/api/auth/forgot-passwordno auth
Запит на відновлення пароля. 3 спроби на 15 хв.
Body: { "email": "string" }
Response 200: { "message": "Якщо email зареєстровано, посилання для відновлення надіслано." }
POST/api/auth/reset-passwordno auth
Скинути пароль за токеном (дійсний 1 год).
Body: { "token": "string", "newPassword": "string" }
Response 200: { "message": "Пароль успішно змінено. Увійдіть з новим паролем." }

📦 Lots

GET/api/lotsoptional auth
Список лотів з фільтрами та пагінацією.
Query: ?page=1&limit=30&category=slug&search=text&sort=ending|price-asc|price-desc|new|popular&minPrice=0&maxPrice=99999&condition=new|used&saleType=auction|buynow&sellerId=1
Response 200: { "lots": [...], "pagination": { "page": 1, "limit": 30, "total": 150, "pages": 5 } }
GET/api/lots/topno auth
Топ-10 лотів за кількістю ставок (завершуються скоро).
Response 200: [ { ...lot, category_name, seller_name, seller_rating, main_image }, ... ]
GET/api/lots/:idoptional auth
Деталі лота з фото, ставками, даними продавця.
Response 200: { ...lot, images: [...], bids: [...], isFavorite: false, category_name, seller_name, seller_rating, seller_reviews, seller_city, seller_since }
POST/api/lotsauth
Створити новий лот. До 10 зображень (multipart).
Body (multipart/form-data): title, description?, categoryId, condition=new|used, saleType=auction|buynow, startPrice, buyNowPrice?, reservePrice?, bidStep?, duration (дні), city?, deliveryMethods?, paymentMethods?, images[] (до 10 файлів)
Response 201: { "message": "Лот створено!", "lotId": 1 }
PUT/api/lots/:idauth
Редагувати лот (тільки без активних ставок).
Body: { "title": "string", "description?": "string", "condition?": "string", "startPrice?": 100, "buyNowPrice?": 500 }
Response 200: { "message": "Лот оновлено" }
DELETE/api/lots/:idauth
Видалити лот (тільки без активних ставок).
Response 200: { "message": "Лот видалено" }
GET/api/lots/user/myauth
Мої лоти (як продавець).
Query: ?status=active|completed|sold|expired
Response 200: [ { ...lot, main_image }, ... ]

💰 Bids

POST/api/bidsauth
Зробити ставку. Anti-sniping: +2 хв до кінця. Автоставки обробляються автоматично.
Body: { "lotId": 1, "amount": 150, "isAuto?": false, "autoMax?": 500 }
Response 201: { "message": "Ставку прийнято!", "bidId": 1, "newPrice": 150 }
GET/api/bids/myauth
Мої ставки (переможні / перебиті).
Response 200: [ { ...bid, lot_title, current_price, end_time, lot_status, lot_image, bid_status: "winning"|"outbid" }, ... ]
GET/api/bids/lot/:lotIdno auth
Історія ставок по лоту.
Response 200: [ { "amount": 150, "created_at": "...", "bidder": "use***" }, ... ]
POST/api/bids/buy-nowauth
Миттєва покупка за фіксованою ціною. Створює purchase.
Body: { "lotId": 1 }
Response 200: { "message": "Покупка оформлена!", "amount": 500 }

🛒 Purchases

PUT/api/purchases/:id/receiveauth
Позначити як отримано (покупець).

POST/api/purchases/:id/reviewauth

Body: { "rating": 5, "text?": "Все супер!" }

💳 Payment

POST/api/payment/createauth
Створити платіж через LiqPay. Повертає data+signature для checkout.
Body: { "purchaseId": 1 }
Response 200: { "data": "base64...", "signature": "base64...", "checkoutUrl": "https://www.liqpay.ua/api/3/checkout?data=...&signature=..." }
POST/api/payment/callbackserver
LiqPay callback (server-to-server). Обробляє оплату автоматично.
Body: { "data": "base64...", "signature": "base64..." }
Response 200: "OK"
GET/api/payment/status/:purchaseIdauth
Перевірити статус оплати.
Response 200: { "id": 1, "status": "paid", "amount": 500, "paid_at": "..." }
POST/api/payment/monobankauth
Створити платіж через Monobank (довідкова інформація).
Body: { "purchaseId": 1 }
Response 200: { "message": "...", "amount": 500, "currency": "UAH", "instructions": [...] }
